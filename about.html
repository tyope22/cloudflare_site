About This Project & Me

Who Am I?

I am a developer who enjoys building fast, simple, and reliable web applications. This site is a personal project exploring modern web development and deployment practices.

The Mission

This entire website is hosted on **Cloudflare Pages**, demonstrating how easy it is to deploy a static site directly from a GitHub repository. The goal is to keep the codebase minimal, the loading speed instantaneous, and the learning curve fun. Every file you see is managed via Git commands run directly from a mobile terminal environment (UserLAnd), proving that you can build and deploy powerful tools from almost anywhere!

Technologies Used

  • **HTML5 & CSS3:** For structuring and styling the content.
  • **Tailwind CSS:** For rapid, utility-first responsive styling.
  • **Git & GitHub:** For version control and source code management.
  • **Cloudflare Pages:** For hosting, continuous deployment (CD), and global edge caching.
  • **UserLAnd:** The mobile Linux environment used to manage the Git repository.
